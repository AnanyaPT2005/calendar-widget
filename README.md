## What it does
Calendar Widget is a lightweight desktop application that displays the **current day and month** in a clean, minimal window.  
It runs offline and behaves like a native Windows app.

## How it was made
- Built using **Electron**
- UI created with **HTML, CSS, and JavaScript**
- Packaged into a Windows app using **electron-builder**

The app is developed locally and then built into an installer so it can run without VS Code or the terminal.
